# Friscoll.id Website

Website statis modern untuk **Friscoll.id**, sebuah brand yang menyediakan koleksi kemeja batik berkualitas dengan desain elegan dan modern. Website ini dibuat sebagai landing page yang menampilkan informasi brand, galeri produk, testimoni pelanggan, serta informasi kontak dalam tampilan yang responsif dan menarik.

## 🌐 Demo

> Tambahkan link website setelah di-deploy.

```
https://friscoll.vercel.app
```

## 🚀 Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/Fatkhur-RZQ/Batik-Friscoll.git
```

2. Masuk ke folder project

```bash
cd Batik-Friscoll
```

3. Jalankan menggunakan Live Server atau buka file `index.html` langsung di browser.
